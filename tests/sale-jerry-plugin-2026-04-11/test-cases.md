# Sale Jerry Plugin 测试用例集 (2026-04-11)

## 概述
本文档包含 sale-jerry-plugin 本次迭代（v2.0 + 兼容性增强）的全部测试用例，涵盖：
- project-init 的重大重写（v2.0）
- 7个下游 Skill 的向后兼容性增强
- 核心功能、边界情况、破坏性变更检测

**测试范围**：
- 目录结构升级：旧结构 → 新结构 + 中间层 `售前阶段/`
- 版本检测与迁移：自动识别旧版、提示升级、执行迁移
- 双路径兼容：新旧路径自动识别与 fallback
- 行业检测与参数新增：customer_industry 三级级联
- 9个新增目录等效映射规则
