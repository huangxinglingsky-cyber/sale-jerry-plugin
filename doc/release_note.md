## v1.3.0 (2026-03-25)

### 更好用了 🔧

- **企业调研** (~) 案例数据同步到最新版本，搜索到的案例信息更全更新，找银行类客户案例时也能自动覆盖所有银行类型
- **招投标分析** (~) 分析用的案例数据升级到最新版本，得分计算和策略建议更准确；资质信息也同步更新了
- **商务控标** (~) 案例数据升级到最新版本，控标分析更精准；资质信息也同步更新了
- **销售准备** (~) 案例数据统一升级，一键准备销售资料时引用的案例都是最新的
- **需求匹配** (~) 功能清单升级到最新版本，匹配客户需求时用的产品功能信息更准确更全面

### 修好了 🛠

- **立项报告** (!) 修复了生成报告时搜索模板路径不正确的问题，现在能正常找到知识库中的模板和参考资料
- **商务控标 + 招投标分析** (!) 修复了两个功能之间数据传递不匹配的问题，分析结果能正确传递给控标报告
- **需求匹配** (!) 修复了需求匹配结果无法正确记录到项目状态的问题
- **销售陪练** (!) 修复了陪练时无法读取项目资料的问题，演练场景会基于真实的会议记录和需求文档
- **话术生成** (!) 修复了话术生成后项目状态更新失败的问题
- **会议分析** (!) 修复了会议分析结果无法正确保存到项目的问题
- **技能质检** (!) 审查功能更聚焦了，不再检查与销售无关的内部工具，审查结果更简洁实用
- **项目初始化** (!) 修复了创建带客户名称的项目时，系统误报"项目不存在"的问题
- **立项报告** (!) 修复了知识库目录找不到时直接报错的问题，现在会灵活处理，找得到就用、找不到也不会卡住
- **企业调研** (!) 修复了内部流程步骤编号混乱的问题，案例匹配和报告生成的流程更清晰

## v1.1.0 (2026-03-17)

### 新能力 ✨

- **技能开发助手** (+) 新增一个"AI 开发 AI"的能力——可以帮你快速开发新功能或改进现有功能，改完后会先给你看改了什么，确认无误才会保存
- **技能质检官** (+) 新增自动检查功能，能发现工具内部的逻辑漏洞并给出修复建议，确保输出结果更可靠

### 更好用了 🔧

- **项目状态更新** (~) 修改项目进度时会自动保留修改记录，方便回溯每次改了什么、谁改的，再也不怕版本混乱
- **插件发布** (~) 发布新版本后会自动通知团队群，大家第一时间知道有哪些新功能可以用

## v1.0.1 (2026-03-15)

### 更好用了 🔧

- **立项报告** (~) 生成报告时只展示和你需求相关的模块方案，不再混杂无关内容，报告更精炼
- **企业调研** (~) 输入公司简称也能自动识别全称，信息不够时会主动问你补充关键细节，确保调研结果准确无误
- **会议分析** (~) 哪怕只有几段简短的对话记录，也能帮你分析出有用的销售线索和推进建议
- **销售话术** (~) 给你写话术时会参考更多真实案例和产品知识，建议更贴合实际场景
- **案例匹配** (~) 找到的案例会标注来源，方便你追溯和核实
- **招投标分析** (~) 信息不完整时会主动提醒你补充关键内容，避免分析结果偏差
- **销售陪练** (~) 练习前会先检查项目信息是否齐全，不完整时会引导你补充，让模拟演练更贴近真实

## v1.0.0 (2026-03-14)

### 新能力 ✨

首次发布销售助手，涵盖 **21 项核心能力**：

#### 销售打单必备

| 能力 | 一句话说明 |
|------|-----------|
| **企业调研** | 输入客户公司名，自动帮你查清这家公司的背景、行业和关键信息 |
| **案例匹配** | 根据行业、产品模块等条件，从历史项目库中帮你找到最相关的成功案例 |
| **销售话术** | 告诉它客户场景，帮你生成专业的销售话术，从开场到促单全覆盖 |
| **销售准备** | 一键搞定企业调研 + 案例匹配 + 话术生成，打单前用它就够了 |

#### 项目跟进利器

| 能力 | 一句话说明 |
|------|-----------|
| **项目初始化** | 新项目一键创建标准文件夹和模板，省去手动整理的麻烦 |
| **项目状态更新** | 随时更新项目进展，自动记录每次修改，多人协作不串位 |
| **相关方录入** | 粘贴一段人员名单就能自动识别姓名、职务、联系方式，批量录入项目成员 |
| **会议分析** | 丢进去会议纪要，自动帮你分析客户需求阶段，并给出下一步怎么推进的建议 |
| **SPIN 分析** | 基于 SPIN 销售方法论，从对话中挖出客户的痛点和需求 |
| **日报生成** | 自动扫描你今天改了哪些文件，一键生成工作日报 |

#### 招投标专项

| 能力 | 一句话说明 |
|------|-----------|
| **招投标分析** | 上传招标评分表，自动计算得分、识别风险点，告诉你哪里能拿分哪里要小心 |
| **商务控标** | 分析招标要求，帮你找出控标机会和优化方向，提高中标率 |
| **需求匹配** | 把客户需求和我们的产品功能逐条对比，算出匹配度，快速判断能不能做 |

#### 销售赋能工具

| 能力 | 一句话说明 |
|------|-----------|
| **销售陪练** | 模拟真实客户和你对话练习，练完还会给你复盘哪里做得好哪里可以改进 |
| **技术黑话翻译** | 客户说了听不懂的技术术语？丢给它，秒变大白话，还帮你对应到自家产品功能 |
| **文档处理** | Word、Excel、PDF 各种格式都能读，客户发来的需求文档直接丢进去分析 |
